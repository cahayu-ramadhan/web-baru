Unsur-Unsur Prisma

  1. Prisma segi-n memiliki:
    • Sisinya ditentukan dengan => n + 2
    • Titik sudutnya ditentukan dengn => 2n
    • Banyak rusuknya ditentukan dengan => 3n
    • Banyak diagonal sisinya => n (n - 1)
    • Banyak diagonal ruang => n (n - 3)

Luas Permukaan Prisma

Untuk menghitung luas permukaan prisma kita harus mengetahui bentuk alas dan tutup dari prisma.

Misalkan:
La = Luas Alas
Ka = Keliling Alas
t = Tinggi Prisma
maka,

LP = (2 × La) (Ka × t)

Volume Prisma

Rumus volume prisma dapat diperoleh dari luas alas dikali tinggi prisma. Jadi,

V = La × t
